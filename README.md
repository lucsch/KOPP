# KOPP

![](art/kopp_icon.png)

## Introduction

Keep track of your timetable and your hours

## Installation

Install the required dependencies using the following command:

```bash
uv sync
```

## Create the version

```bash
uv run kopp/createversion.py version.py
```

## Running the app

Run the app using the following command:

```bash
uv run python -m kopp
```
